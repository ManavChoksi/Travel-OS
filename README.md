# Travel OS 🌍

**Smart Travel Advisor for Indian Families**  
AI-powered trip planner that compares HDFC Infinia SmartBuy, airline miles, hotel points and OTA cashback — so you always book at the best effective cost.

## Features
- 3-step trip planner (dates · preferences · payment arsenal)
- Compares HDFC Infinia SmartBuy 10X vs direct airline portal vs OTA
- Ranked booking strategies with real INR savings
- Day-by-day AI itinerary with timings & meal suggestions
- Hotel recommendations with best booking method
- Booking roadmap with exact platforms & cards to use
- Insider tips & watch-out warnings

## Tech Stack
- React 18 + Vite
- Claude claude-sonnet-4-5-20251001 (AI planning engine)
- Lucide React icons
- Google Fonts (Cormorant Garamond + DM Sans)

## Deploy on Vercel
[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/ManavChoksi/Travel-OS)

1. Click the button above — or import the repo at [vercel.com/new](https://vercel.com/new)
2. No env vars needed (Anthropic key is injected at runtime via Claude's artifact API)
3. Hit **Deploy** — done in ~60 seconds
